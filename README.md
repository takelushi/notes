# Notes

個人的なナレッジベースリポジトリです。学習メモ、技術ノート、参考資料などを整理して保存しています。

## 構成

- `journal/` - 日々の記録
- `notes/` - トピック別のナレッジノート

## 使用ツール

- [Foam](https://foambubble.github.io/foam/)
  - タグとリンクでノートを管理しています。

## ノートの書き方

VS Code のコマンドパレットを使って作成します。

- Journal: `Foam: Open Daily Note`
- Notes: `Foam: Create New`

詳細な規約は [CLAUDE.md](./CLAUDE.md) を参照してください。
